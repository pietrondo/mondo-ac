<Specification: Rotating Lighthouse Beam>
## Description
Defines the rotating volumetric spotlight beam generated by lighthouses.

## Requirements
1. **Volumetric Spotlight**:
   - The lighthouse MUST render a volumetric spotlight beam using a semi-transparent, additive-blended cone mesh pointing outwards.
2. **Dynamic Rotation**:
   - The beam MUST rotate continuously around the vertical Y-axis at a constant speed in the update loop.

## Scenarios
### Scenario 1: Spotlight beam rotation
* **Given** a lighthouse with a volumetric beam mesh is spawned
* **When** the world updates
* **Then** the beam mesh's Y-rotation MUST increase over time
